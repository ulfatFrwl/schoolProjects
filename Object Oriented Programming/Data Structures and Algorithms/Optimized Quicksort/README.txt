Lab 2: Optimized Quicksort

------------------------------------------------------------------------
DESCRIPTION

This is an optimized version of quicksort. The pivot can be the first element
of an array or a median value to prevent ineffective sort where a smallest/largest
element is chosen as a pivot. The program also switches to insertion sort when
a subarray has 0 to 30 elements.
