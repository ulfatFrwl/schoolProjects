Lab 3: ASCII Forest

--------------------------------
DESCRIPTION

In this lab, the user enters two integers x and y.
The program will print x trees with size y.

--------------------------------
FILES

Lab3.asm

This file includes the assembly code of the lab.

--------------------------------
INSTRUCTIONS

This program is intended to be run using the MIPS Assembler and Runtime Simulator (MARS).
Enter the test case as a program argument and run using MARS.
Note: x must be greater than 0 and y must be greater than 1
