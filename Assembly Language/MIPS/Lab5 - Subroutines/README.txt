Lab 5: Subroutines
------------------------------------------------------------------------
DESCRIPTION

3 program arguments are converted from ASCII string to integer then converted
back to individual ASCII character to print on screen using syscall 11.
Note: To increase the challenge, only syscalls 4 and 11 are permitted.

------------------------------------------------------------------------
FILES

-
Lab5.asm

This file includes the assembly code of the program.

-
TestFile.asm
This file is the test file for entering arguments and testing program.

------------------------------------------------------------------------
INSTRUCTIONS

This program is intended to be run using the MIPS Assembler and 
Runtime Simulator (MARS). Enter the test case as a program argument
and run using MARS in the following formats:

0xABCDEF00 0xFFFFFFFF 0x00000010
