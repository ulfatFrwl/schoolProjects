Lab 4: Sorting Floats

------------------------------------------------------------------------
DESCRIPTION

In this lab, 3 arguments are accepted in IEEE 754 single precision
floating point format. These numbers will be numerically sorted in
ascending order, then printed to the screen in order in floating point
format and decimal format.

------------------------------------------------------------------------
FILES

-
Lab4.asm

This file includes the assembly code of the lab.

------------------------------------------------------------------------
INSTRUCTIONS

This program is intended to be run using the MIPS Assembler and 
Runtime Simulator (MARS). Enter the test case as a program argument
and run using MARS.
